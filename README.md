# CSAS 2026 Introduction to R Workshop

## Outline
R is a programming language favored by data scientists for its statistically minded design. This workshop introduces R to those with little or no prior experience. We will cover 
basic R syntax, data structure, and some basic practical uses of R using locally created data as well as a preexisting basketball dataset.

## Prerequisites
A computer with R and Positron/RStudio (Positron suggested) downloaded.

## Instructor
Tyler Adams is a Statistics first-year undergraduate student at the University of Connecticut. He currently maintains the Uconn Data Science website and gained previous experience in R and data science by taking the class DSDA 1010 Data Science Society Using R last semester.
